Advent of Code

Usage:
  aoc -d <day> | --day <day>
  aoc -l | --latest
  aoc -a | --all
  aoc -h | --help

Options:
  -h --help                    Show this screen.
  -l --latest                  Run the most recent challenge day.
  -a --all                     Run all days in order.
  -d <day>, --day <day>  Run a specific day.
